# Lion's Guard Extras

## Description

Adds two new ships to the Lion's Guard's fleet:

* A Conquest Mk II (LG), a Conquest retrofitted with a spinal-mounted Thermal Pulse Cannon.
* A Champion (LG), with some small changes to weapon mounts.
